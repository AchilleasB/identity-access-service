# Identity Access Service

## Overview
The Identity Access Service is a Go-based application designed to manage user login and registration. It provides a secure way to handle user data and interactions through a RESTful API.

## Setup Instructions
1. **Clone the repository:**
   ```
   git clone https://github.com/AchilleasB/baby-kliniek/identity-access-service.git
   cd identity-access-service
   ```

2. **Install dependencies:**
   Ensure you have Go installed, then run:
   ```
   go mod tidy
   ```

3. **Run the application:**
   ```
   go run cmd/api/main.go
   ```
